# OOP in Practice

Repositório criado para praticar conceitos de Programação Orientada a Objetos (POO) utilizando Java.

## Objetivo

Este projeto reúne exercícios e desafios desenvolvidos durante meus estudos, com foco na aplicação prática dos principais conceitos de orientação a objetos.

## Conceitos praticados

* Classes e objetos
* Encapsulamento
* Herança
* Polimorfismo
* Abstração
* Construtores
* Coleções
* Organização de pacotes
* Boas práticas de programação em Java

## Estrutura do projeto

O projeto está organizado em pacotes separados por desafios:

```text
src/
└── br.com.alura
    ├── challenge1
    ├── challenge2
    ├── challenge3
    ├── challenge4
    └── challenge5
```

Cada pacote contém exercícios e implementações desenvolvidas para reforçar conceitos específicos da linguagem e da orientação a objetos.

## Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git
* GitHub
